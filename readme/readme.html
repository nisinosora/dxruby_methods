HP-Gage

HpGage.new(x:, y:, width:, height:, hp:, maxhp:, color:, bgcolor:, alpha:, bgalpha:)
変数説明デフォルト値
x:, y:ゲージを表示させる位置を指定します。0
width:ゲージの横幅を指定します。100
height:ゲージの高さを指定します。10
hp:初期値を指定します。100
maxhp:最大値を指定します。100
color:ゲージの色を指定します。C_WHITE(白)
bgcolor:ゲージの背景色を指定します。nil(指定なし)
alpha:, bgalpha:不透明度を指定します。100

draw

ゲージを表示します。

Hit

ob1, ob2
どちらも、Spriteクラスを使います。
Spriteクラスが入っている配列でも問題ありません。

Hit.check(), Hit.check_index(), Hit.check_with_index()
それぞれブロック処理を行いますが、ブロックにより渡す値が変化します。
なお、2つの引数が両方ともSpriteクラスの場合、値を渡すことはありません。

Hit.check()は、両方が配列になっている場合、それぞれのオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側のオブジェクトを渡します。

Hit.check_index()は、両方が配列になっている場合、それぞれの要素番号を渡します。
どちらか片方が配列になっている場合、その配列側の要素番号を渡します。

Hit.check_with_index()は、両方が配列になっている場合、それぞれの要素番号とオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側の要素番号とオブジェクトを渡します。

BGM

音楽ファイルです。

wavかmidファイルを入力してください。

play, stop

playは、再生です。

stopは、停止です。

SetInterval

何フレームごとに処理を行うかを判断するため、秒数を入力してください。

loop

ブロックで、行いたい処理をブロックの中に記入します。
