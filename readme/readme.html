HP-Gage

HpGage.new(x:, y:, width:, height:, hp:, maxhp:, color:, bgcolor:, alpha:, bgalpha:, direction:)
変数説明デフォルト値
x:, y:ゲージを表示させる位置を指定します。0
width:ゲージの横幅を指定します。100
height:ゲージの高さを指定します。10
hp:初期値を指定します。100
maxhp:最大値を指定します。100
color:ゲージの色を指定します。C_WHITE(白)
bgcolor:ゲージの背景色を指定します。nil(指定なし)
alpha:, bgalpha:不透明度を指定します。([透明]0～255[不透明])255
direction: ゲージの動く向きを指定します。("normal" か "reverse" か "both")
"normal"は、通常でhpが少ないほど、左側に減っていきます。
"reverse"は、hpが少ないほど、右側に減っていきます。
"both"は、ゲージの両端から変動します。
具体的な動きは、サンプルをご確認ください。 		nil
表示する際に指定がなければ、"normal"として表示します。
directionで先に指定されている場合は、そちらが優先されます。

draw(direction)

ゲージを表示します。
directionを指定すると、ゲージの動く向きが変わります。
"normal"は、通常でhpが少ないほど、左側に減っていきます。
"reverse"は、hpが少ないほど、右側に減っていきます。
"both"は、ゲージの両端から変動します。
具体的な動きは、サンプルをご確認ください。

Hit

ob1, ob2, bool = true
「ob1, ob2」どちらも、Spriteクラスを使います。
Spriteクラスが入っている配列でも問題ありません。

「bool」は、trueかfalseを指定します。
trueの時は、当たっているとき
falseの時は、当たっていないとき

Hit.check(), Hit.check_index(), Hit.check_with_index()
それぞれブロック処理を行いますが、ブロックにより渡す値が変化します。
なお、2つの引数が両方ともSpriteクラスの場合、値を渡すことはありません。

Hit.check()は、両方が配列になっている場合、それぞれのオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側のオブジェクトを渡します。

Hit.check_index()は、両方が配列になっている場合、それぞれの要素番号を渡します。
どちらか片方が配列になっている場合、その配列側の要素番号を渡します。

Hit.check_with_index()は、両方が配列になっている場合、それぞれの要素番号とオブジェクトを渡します。
どちらか片方が配列になっている場合、その配列側の要素番号とオブジェクトを渡します。

BGM

音楽ファイルです。

wavかmidファイルを入力してください。

play, stop

playは、再生です。

stopは、停止です。

SetInterval

何フレームごとに処理を行うかを判断するため、秒数を入力してください。

loop

ブロックで、行いたい処理をブロックの中に記入します。

TextSelect

TextSelect.new(x:, y: text:, size:, color:, bgcolor:, bgalpha:, hover:, font_alpha:)

変数説明デフォルト値
x:, y:文字を表示させる位置を指定します。0
text:表示する文字を指定します。"sample"
size:文字サイズを指定します。20
color:文字色を指定します。C_WHITE(白)
bgcolor:背景色を指定します。C_BLACK(黒)
bgalpha:背景の不透明度を指定します。([透明]0～255[不透明])0
hover:マウスカーソルが文字に重なった時の、文字の色を指定します。nil
font_alpha:文字の不透明度を指定します。([透明]0～255[不透明])255

draw
文字を表示させます。

check()
マウスが文字に重なった時の処理を書きます。
引数で(false)を指定すると、重なってない時に実行します。

draw_check
上記の処理をまとめて行います。
この処理の利用頻度は、高いと思います。
引数で(false)を指定すると、重なってない時に実行します。

BackGround

BackGround.new(scale_x, scale_y, 画像ファイル)

scale_xの位置には、画像の横幅を指定します。１が通常サイズです。
scale_yの位置には、画像の縦幅を指定します。１が通常サイズです。
画像ファイルは、pngやjpgなど背景画像として利用するファイルを指定します。

draw
背景画像を表示します。
複数の画像を設定する場合、最後に適応した画像に応じて画面の
大きさが変わります。場合によっては、大きすぎたり小さすぎたりするため、ご注意ください。

Boolean

初期のブール値(true, false)を指定してください

デフォルト: true

chenge()

trueの時はfalseに、falseの時はtrueに切り替えます。
引数(trueかfalse)が指定されている場合、そのブール値になります。
なお、true・false以外の値が入った場合、強制的にtrueにしています。
